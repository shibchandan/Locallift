# Project Summary
LocalLift is a MERN-stack and AI-powered web platform designed to connect individuals and small business owners with nearby delivery persons on demand. The platform enables users to easily request delivery services for various items, from groceries to documents, facilitating quick and efficient local deliveries. LocalLift empowers delivery persons and small businesses by providing a flexible and user-friendly interface to manage delivery requests, negotiate prices, and communicate seamlessly.

# Project Module Description
- **User Authentication**: Allows users to register, log in, and manage their accounts.
- **Delivery Request Management**: Users can create, view, and manage delivery requests.
- **Real-Time Communication**: Integration with WhatsApp for sharing product images and details.
- **Price Negotiation**: Users can negotiate delivery costs through built-in chat or calls.
- **AI Features**: Includes smart price suggestions and an AI matching system for delivery persons.

# Technology Stack
- **Frontend**: React.js, Tailwind CSS, Vite
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **AI Microservice**: Python (Flask or FastAPI) with Scikit-learn or TensorFlow
- **Real-Time Communication**: Socket.io
- **Messaging Integration**: Twilio or WhatsApp Business API

# Usage
To set up and run the project:
1. Install dependencies: `pnpm install`
2. Run the linter to check for issues: `pnpm run lint`
3. Build the project: `pnpm run build`
4. Start the development server: `pnpm run dev`
